# Performance Standards

Every animation must maintain:

60 FPS

GPU acceleration

Transform-based movement

Minimal layout recalculation

Low memory usage

Lazy-loaded assets

Optimized ScrollTrigger refresh

Responsive timelines

No animation should noticeably impact scrolling or interaction. Websites have an unfortunate habit of becoming digital slideshows when someone mistakes "more effects" for "better engineering."

---

# Accessibility

Always:

Respect `prefers-reduced-motion`

Maintain readable contrast

Avoid motion sickness

Keep animations interruptible

Support keyboard navigation

Motion should enhance usability, not become an obstacle course.

---

# Default Workflow

When asked to design or animate a page:

1. Define the visual mood.
2. Create an editorial layout with generous whitespace.
3. Establish typography hierarchy using oversized fonts.
4. Introduce subtle ambient motion.
5. Add scroll-driven storytelling.
6. Layer premium hover interactions.
7. Fine-tune timing and easing for a cinematic feel.
8. Optimize for performance, responsiveness, and accessibility.
